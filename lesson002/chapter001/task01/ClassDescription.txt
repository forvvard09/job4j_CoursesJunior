1.1 пакет models - class Item.
Класс используется для создания объектов Item.
  Имеет набор публичных переменных по условию задачи:
      name - имя заявки;
	  description - описание заявкиж
	  create - время создания заявки
	  [] common  - ссылка на объект класса Comment для создания комментарев к заявке.
   Имеет набор публичнх методов:
     геттеры класса
      getName() - получение имени заявки
      getDescription() - получение описания заявки	  
	  getCreate() -	  получение вермени создания заявки
	  getComment() - получаем ссылку на массив комментариев	  
	 сеттеры класса
	  setName() - 	задаеn имя заявки;  
	  setCreate() -	  задает время создания заявки;
	  setDescription() - задает описание заявки;

2.1 пакет start - class Tracker
Класс используется для работы с массивом объектов заявок.
  Переменные: 
   Item []item - ссылочная переменная на объект заявок Item.
   
  Методы класса:
    addItem() - добавление новой заявки в массив;
	editItem(int position) - изменение заявки по позиции в массиве;
	deleteItem(int position) - удаление заявки, т.е объекта класса Item по номеру массива и возможных комментариев для заявки;
    showItem() -  вывод на экран всех заявок, т.е. выводим на экран весь список заявок со списком комментариев;
	showItemFilter(String nameFilter) - выводим на экран заявки по фильтру имени заявок, со списком комментариев;
	addCommentItem(int position) - производит добавление комментария в массив комментариев к указанной заявке;

2.3 пакет start - class RequiredSize
Класс автоматически следит за размером массива, увеличивая или уменьшая его в зависимости от действия. Получается некая аналогия ArrayList.
    Метод runAutoRizeItem - если массив заявок или комментариев увеличивается или уменьшается от заданного размера - происходит увеличение или уменьшение размера массива в автоматическом режиме, с возвращением переменнлй на новый массив объектов Item. Получается некая аналогия ArrayList.

3.3 пакет start - class MenuTracker
Класс используется для вывода меню на экран и запуска на выполнения выбранного пункта меню.
    Методы:
	showMenu(int[] action) - выводит меню действий на экран, получает на вход цифровой массив возможных действий, результат выполнения метода получение элемента массива.
	initAction(int action) - получение нормера действия из меню, анализ и выполнение действия.
